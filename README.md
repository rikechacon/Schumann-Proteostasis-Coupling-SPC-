# Schumann-Proteostasis Coupling (SPC)

> Modelo teórico-computacional del acoplamiento entre resonancias electromagnéticas terrestres (Schumann) y la dinámica de proteostasis neuronal.

## 🔬 Resumen Científico

Este proyecto explora la hipótesis de que los campos electromagnéticos de baja frecuencia, específicamente las Resonancias de Schumann (~7.83 Hz), pueden modular la estabilidad proteostática en neuronas mediante mecanismos de transducción mediados por agua vicinal y membranas celulares.

El marco matemático se basa en:
- Ecuaciones de Langevin con potencial de doble pozo (Landau-Ginzburg)
- Acoplamiento estocástico forzado por señales EM externas
- Métricas de estabilidad: Mean First Passage Time (MFPT)

**Palabras clave**: neurociencia teórica, biofísica, resonancia estocástica, proteostasis, campos electromagnéticos, agua vicinal.
